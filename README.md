# API-Testing-Postman-Project
Postman API Testing Project including User Registration, Login, Category Management and Token Authentication.

This project demonstrates API testing using Postman including :-

1. User Registration
2. User Login
3. Admin Login (Get Token)
4. Add Category
5. View Category
6. Delete Category

## Tools Used
- Postman
- REST API
- GitHub
